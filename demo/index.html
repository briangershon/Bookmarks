The situation

Browser bookmarks are an awesome feature that help you manage all web pages that you would like to revisit in future. I have a 9 to 5 job where during the day I'm too busy to read all the interesting web pages that I visit, so I just bookmark them.

The problem

When I get home, where I have free time to surf the internet, I cannot access all those awesome hacker stories that I bookmarked during the day.

Possible solution

Googling the web brought up a number of bookmarking services like Netvouz, Ma.gnolia, and Del.icio.us but none of them was quite right for my simple need, "just to carry my bookmarks" wherever I go.


After much deliberation I finally decided that if I needed a tailored bookmarking service, I was going to build it myself (troubles of being a self-proclaimed hacker) - here is my solution.

The Application

A hacker's bookmarks is an application written in 'vanilla' JavaScript and stores all bookmarks in a flat file database - a Google docs spreadsheet. To retrieve bookmarks from the database (Google docs spreadsheet), the application creates a YQL query that returns data in JSON format.

Step 1: Create a Google docs form

Create a form in google docs with two text fields, one for the bookmark url and the other for the bookmark title, respectively.

Save your form and click the link at the bottom of the page to view the newly created form.

 

Step 2: Generate a link for bookmarking

Open the source file of the form and copy the URI in the action attribute
The copied URI looks like the one below :

https://spreadsheets.google.com/formResponse?formkey=dFhsQlBxQmFsSnNvaVpQS3doSzM2MlE6MQ&ifq

Remove the ifq field from the URI so that you are only left with the part similar to the one shown below :

https://spreadsheets.google.com/formResponse?formkey=dFhsQlBxQmFsSnNvaVpQS3doSzM2MlE6MQ

Paste your edited URI below to generate a bookmarking link and drag it to your bookmarks

Step 3: Generate a link for viewing bookmarks

Go back to the google docs home page and open the spreadsheet for the form. This is your flat file database, all form input is stored here.

Now you need to configure your database (Google docs spreadsheet).

Firsly change file permissions to Public on the web and then on the top-right corner of the page click on the Share arrow.
